Reflection: Your thoughts, comments and lingering questions about this product/service: I think this is a great piece of software. I especially like the bimodal approach of being able to see what is being read as it is being read. I think the software is overpriced for what it does. While it certainly does what it claims to do very well, there are ways to get text read to you for free. The highlighting is a more difficult feature to replicate but it would be hard to justify the additional cost. A question that this brings up is how effective the bimodal approach is over just standard reading of the text. I guess some kind of data on the effect of the highlighting would be great to see. That might help justify the cost. If it wasn’t so expensive it would be a great piece of software because it is so versatile and does include some small features like the note-taking and flashcards that I could see being helpful. Also, the network installation option is beneficial because that means that they software doesn’t have to be locked down to any one computer and can be used anywhere.
