FTP stands for File Transfer Protocol. It is a way to upload files to a server
where they can be stored and accessed by other FTP users who have the
right permissions and know the server’s address.

FTP uses the internet as a
gateway to send and receive the files. Servers that host files uploaded via FTP
can sometimes be accessed via a web browser, or they can be accessed using
an FTP client like FileZilla.

A ListServ is a list that
can both distribute emails out to a group of users, and
act as a message board for users to post messages to. For example, here at
Harvard Business School we use listservs a lot for providing support for
certain events. The listerv allows us to have studen
ts email one address and
have multiple people receive the message. The message is also stored so that
it can be referenced later. Another great feature of listservs is that they can
have subscriptions. That means that the users can be managed both from a
“
who can send to this list” perspective, and “who received messages sent to
this list” perspective.
